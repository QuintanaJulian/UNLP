DISCOS
los discos estan particionados en

Sectores: Unidad de almacenamiento, donde se guarda la informacion

Cluster
	Conjunto de sectores consecutivos (siemrpe multiplos de 2 sectores).
	Puede o no usarse, si existe esta es la unidad minima.
	Permite almacenar archivos mas grandes guardando menos direcciones de memoria.
	Cuanto mas grande el cluster mas eficiente para archivos mas grandes, pero mayor fragmentacion interna para archivos chicos.
	
File System
	Formato en el que los archivos estan almacenados


FAT: File Alication Table
	Contien la informacion sobre en que lugar estan almacenados los archivos
