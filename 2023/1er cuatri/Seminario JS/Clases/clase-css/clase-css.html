Cascading Style Sheets - CSS

  • Describe como el contenido de la página Web va a ser visualizado. Permite ahorrar muchísimo trabajo para visualizar con el mismo estilo a todas las páginas del sitio Web, en distintos dispositivos y tamaños de pantalla.
  • Las hojas de estilos se almacenan en archivos .css externos. Un código HTML no debe tener código CSS.
  • Estandard definido por la W3C.

Reglas CSS

Consiste de un selector y un bloque de declaraciones.

Sintaxis CSS

Reglas CSS

  • Selector: selecciona el elemento HTML a aplicar el estilo.
  • Bloque de declaraciones {}: una o más declaraciones separadas por ;
  • Cada declaración incluye una propiedad, nombre y valor separadas por ,

Ejemplo de una regla CSS


                            p {
                                color: red;
                                text-align: center;
                              }
Selector: p
Propiedades: color y text-align
Valores de la propiedad: red y center
Probarlo

Selectores CSS

Un selector se usa para encontrar el elemento HTML al cual aplicarle el estilo. Categorías:

  • Simples: por id, name, class.
  • Combinados: elementos relacionados.
  • Pseudo-clase: basado en ciertos estados.
  • Pseudo-elementos: aplica un estilo a una parte del elemento.
  • Atributo: basado en un atributo o en el valor de un atributo.
Ver ejemplo de selectores simples.

Selectores CSS. Universal y Agrupamiento

* : selecciona todos los elementos HTML de la página. 


                            * {
                                text-align: center;
                                color: blue;
                              }
                        

                            h1, h2, p {
                                text-align: center;
                                color: red;
                              }

Selectores Simples

Selector Ejemplo Selecciona a:
#id #par1 Selecciona el elemento p con id par1
.class .descripcion Selecciona todos los elementos con la clase descripcion

Selectores simples - Continuación

Selector Ejemplo Selecciona a:
element.class p.descripcion Todos los p con clase intro
element h1 Todos los elementos h1
element,element h1, h2, .. Todos los elementos h1 y h2

  • external: en un archivo css. No debe contener tags HTML.
  • en línea con el tag HTML

Colores

Soportados por la mayoría de los navegadores, se especifican a través de:

  • Palabras claves (140 nombres estandard)
  • RGB: (red, green, blue)
  • HEX: valor hexadecimal
  • RGBA: RGB + canal alpha de transparencia de la imágen
  • HSL y HSLA: indican tono, saturación y luminosidad

CSS - Propiedades Backgrounds

Se utilizan para efectos en el fondo de los elementos.

  • background-color
  • background-image
  • background-repeat: repeat-x / repeat-y
  • background-attachment: scroll / fixed

CSS - Propiedades Backgrounds

Background Position

Relacionado con background-image, es posible especificar la ubicación de una imágen.
Valores posibles: rigth, left, center seguido de top, center, bottom. Ej. right top

Modelo de cajas

Cada elemento HTML posee ciertas dimensiones manipulables a través de CSS:

Ancho y alto de un elemento

Ancho Total = width + left padding + right padding + left border + right border + left margin + right margin
Alto Total = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

Herencia CSS

Ciertos elementos pueden heredar ciertas propiedades de padres a hijos, como color y font-family, modificando el comportamiento por defecto.
border no se hereda. 


                            .container {
                                color: green;
                                border: 1px solid red;
                              }
                              
                              .child {
                                color: inherit;
                              }

Maquetación y layouts

Los elementos HTML tienen "tipo de representación visual": inline y block
Pueden modificarse a través de CSS. Ver ejemplo display

Cascada CSS

Conjunto de reglas y normas del navegador para aplicar estilos CSS a un elemento.
El tema es cuando se superponen los estilos para un mismo elemento, ¿cuál aplica? En general la última definida aunque puede haber salvedades respecto a importancia, origen, orden de aparición y especificidad.

BootStrap CSS

Framework para el desarrollo de sitios Web responsivos.
Brinda un conjunto de plantillas, tipografías, botones, etc.
Sólo para front-end
